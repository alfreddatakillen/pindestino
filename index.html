Pindestino

Pindestino is a node.js kiosk mode Raspberry Pi distribution

Download this project as a .zip file Download this project as a tar.gz file

Features

  • Pindestino will boot from a read-only SD card.
  • At boot, it will run your node.js web app from an USB.
  • When your node.js server has started and listens on port 80 on localhost, Pindestino will start a web browser in full screen kiosk mode and point it to your web app.

Audience

This distribution was first made for running information screens at the Royal Dramatic Theatre in Stockholm.

It can be used for information screens, adscreens, kiosk computers, etc.
